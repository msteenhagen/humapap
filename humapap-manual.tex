\section{Introduction}
The \LaTeX~system is used in various disciplines within the humanities. 
The \verb+humapap+ document class is a straightforward but convenient development of the standard \verb+article+ class. It is specifically designed to meet some of the needs of researchers working in the humanities.

\section{Usage}
The class can be used simply by calling for it in a standard way. Note that the \verb+HumaPap+ class makes use of the \verb+\documenttitle+ option. 
\begin{quote}
	\verb+\documentclass{humapap}+\\
	\verb+\begin{document}+\\
	\verb++\\
	\verb+\documenttitle+\\
	\verb++\\
	\verb+Lorem ipsum dolor sit amet, consectetur adipisicing elit.+\\
	\verb++\\
	\verb+\end{document}+
\end{quote}

\subsection{Class options}
Class options can be used to alter the default document layout. Simply add options to the \verb+\documentclass+ command in the document's preamble:
\begin{quote}
		\verb+\documentclass[option-1, option-2, ...]{humapap}+
\end{quote}
